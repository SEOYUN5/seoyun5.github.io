Portfolio Website

Freelancer 테마를 활용하여 GitHub Pages로 배포한 자기소개용 포트폴리오 웹사이트입니다. HTML, CSS 기반으로 제작되었으며, 테마 커스터마이징을 통해 본인의 정보로 구성하였습니다.

Chat Server

Python 소켓 프로그래밍을 이용한 채팅 서버 구현 프로젝트입니다. 닉네임 설정, 접속자 목록, 브로드캐스트 기능 등을 포함할 예정입니다.

Login System

사용자 인증 기능이 포함된 로그인 시스템으로, HTML과 JavaScript를 활용하여 프론트엔드를 구성하고, LocalStorage를 활용한 로그인 상태 유지 기능을 구현할 예정입니다.

Online Shop Backend

쇼핑몰 백엔드 서버를 구성하기 위한 프로젝트로, 상품 목록 조회, 장바구니, 주문 처리 등의 기능을 설계하고 Java 또는 Python으로 구현할 계획입니다.

Cloud Storage

클라우드 업로드 기능을 프론트엔드 형태로 구현할 예정입니다. HTML과 JS를 활용하여 간단한 업로드 인터페이스를 만들고, 시나리오 기반으로 작동하도록 구성할 계획입니다.

Database 연결

C와 데이터 저장 또는 외부 DB 연동 기능을 포함하는 시나리오를 기반으로 구현할 예정입니다. 추후 백엔드 서버와 연동하여 데이터 CRUD 기능을 개발할 예정입니다.
